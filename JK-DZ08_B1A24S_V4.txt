Host sends: frame ID 01 ;
data 0xF0 0x10 ; // set number of monomers 16 string

EQ response ID : 01 ;
Response data: F1 10 ; // number of monomers 0x10 = 16 String, the transmission data is consistent with the received data, the setting is successful

Host sends: frame ID 01 ;
data 0xF0 0x20 ; // set number of monomers 32 string (Out of range)

EQ response ID : 01 ;
Response data: F1 10 ; // number of monomers 0x10 = 16 string, Transmission data and reception data inconsistency, setting failed

Setting Success

EQ response ID : 01 ;
